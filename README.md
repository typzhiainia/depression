# depression

> 心理健康自助测评平台 —— 基于 PHQ-9、GAD-7、PSS-10 等国际标准量表，提供免费、匿名、即时的心理状态自我评估。

## ✨ 功能特性

- 📋 **标准量表** — 内置 PHQ-9（抑郁）、GAD-7（焦虑）、PSS-10（压力）等国际通用心理测量工具
- 🔒 **完全匿名** — 不收集个人身份信息，数据存储于本地 SQLite 数据库
- ⚡ **即时反馈** — 提交后立即生成分析报告与分级建议
- 📊 **历史记录** — 支持查看过往测评记录，追踪心理状态变化趋势
- 📱 **响应式设计** — 适配桌面端与移动端，随时随地完成测评
- 👨‍💼 **管理后台** — 提供管理员面板，查看统计数据与管理测评记录
- 🌐 **中文界面** — 全站中文化，贴合国内用户使用习惯

## 🛠️ 技术栈

| 技术 | 说明 |
|------|------|
| PHP | 后端逻辑 |
| SQLite | 轻量级数据库，无需额外配置 MySQL |
| HTML/CSS/JS | 前端页面与交互 |
| PDO | 数据库操作（参数化查询，防 SQL 注入） |

## 🚀 快速开始

### 环境要求

- PHP 7.4+
- 启用 `pdo_sqlite` 扩展
- Web 服务器（Apache / Nginx 或 PHP 内置服务器均可）

### 安装步骤

1. **克隆或下载本项目到 Web 目录**

2. **运行安装脚本初始化数据库：**
   ```bash
   php install.php
   ```
   或浏览器访问 `/install.php`

3. **启动服务：**
   ```bash
   cd depression
   php -S localhost:8080
   ```

4. **访问** `http://localhost:8080` 即可使用
